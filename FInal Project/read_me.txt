This is a Fantasy Cricket game created by Het Patel

The main.py file is the main project which imports evaluate.py from evaluate folder
(Note: DO NOT DIRECTLY RUN evaluate.py JUST RUN main.py AND IT WILL IMPORT NECESSARY MODULES)

cric_game.db is the database which contain all important data

main.ui and evaluate.ui are 2 ui files used in this project

Pics folder contains the icons used in the project to make it look good.
